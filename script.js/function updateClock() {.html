function updateClock() { const now = new Date(); const hours = String(now.getHours()).padStart(2, '0'); const minutes = String(now.getMinutes()).padStart(2, '0'); const seconds = String(now.getSeconds()).padStart(2, '0'); document.getElementById('clock').textContent = `${hours}:${minutes}:${seconds}`; } // Atualiza o relógio a cada segundo setInterval(updateClock, 1000); updateClock(); // atualiza imediatamente ao carregar // Alternância de tema claro/escuro document.getElementById('toggleTheme').addEventListener('click', () => { document.body.classList.toggle('dark'); });
